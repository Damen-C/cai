Project No.2

Topic Based Sentiment Analysis on Qatar World Cup Twitter Data

The 2022 Qatar World Cup was full of surprises! From Saudi Arabia shocking the world by upsetting Argentina to Morocco's historic run to the semifinals, you must heard or witnessed those moments during that soccer craze. In this project, I use BERTopic to model topics on tweets posted during the World Cup 2022. Then I use Hugging Face API to conduct sentiment analysis on those tweets. Finally, I connect the topic modeling and sentiment analysis to conduct topic based sentiment analysis and visualize the result.

Project No.3

Using BERTopic to Analyze Japanese Texts

A notable trend among BERTopic usage examples is that they primarily focus on English language texts. This is because the English language typically employs spaces to separate individual words, making it easier for BERTopic to analyze the text. However, languages like Chinese and Japanese use a different system to denote word boundaries, which can pose significant challenges when using BERTopic for topic modeling. In this project, I highlight these disparities and showcase the impact of using a tokenizer when analyzing Japanese texts for topic modeling with BERTopic. By comparing the results generated with and without a tokenizer, I illustrate how tokenization significantly affects the outcome of topic modeling, particularly in languages like Japanese where proper segmentation of words is crucial.
